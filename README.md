# 古诗词展示应用

这是一个展示唐诗宋词的网页应用，支持分类浏览和搜索功能。

## 功能说明

1. **分类浏览**：左侧导航栏可以切换唐诗和宋词分类
2. **诗歌详情**：点击具体诗歌查看详情，包括标题、作者和内容
3. **搜索功能**：可以根据标题或作者搜索诗歌
4. **视频展示**：在诗歌详情页面，2/3区域展示相关视频，1/3区域展示诗词内容

## 视频功能使用说明

1. **添加视频文件**：
   - 在项目根目录创建一个`videos`文件夹
   - 将每个诗词对应的视频文件放入该文件夹
   - 视频文件名建议使用诗词标题，例如`咏鹅.mp4`

2. **关联视频与诗词**：
   - 打开`app.js`文件
   - 在`showPoemDetail`函数中，修改视频源的设置：
     ```javascript
     // 示例：使用诗词标题作为视频文件名
     videoElement.src = `videos/${poem.title}.mp4`;
     ```
   - 确保视频文件名与诗词标题完全匹配（注意大小写和特殊字符）

3. **视频格式支持**：
   - 建议使用MP4格式，兼容性最好
   - 其他支持的格式：WebM, Ogg

## 响应式设计

- 在桌面设备上，视频和诗词并排显示（2/3 + 1/3）
- 在移动设备上，视频和诗词垂直显示（视频在上，诗词在下）
